## Practice 5.3.6

1. 完全ベータ簡約

t1 -> t1'
---------
t1 t2 -> t1' t2


t2 -> t2'
---------
t1 t2 -> t1 t2'


(\x.t12)t2 -> [x -> t2]t12


t1 -> t1'
------------------
(\x.t1) -> (\x.t1')


2. 正規順序

t1 -> t1'
---------
t1 t2 -> t1' t2


(\x.t12)t2 -> [x -> t2]t12


t1 -> t1'
------------------
(\x.t1) -> (\x.t1')


と考えたのだが、最初のルールだと少し雑で、例えば

x ((\y.y)(\z.z))

のときに、　(\y.y)(\z.z) が簡約されない(正規順序では最も左で最も外の簡約基を簡約する決まりになっている)。

そこで、以下のように場合分けして、

(抽象正規形)
nf := \x.nf
      nanf

(非抽象正規形)
nanf := x
        nanf nf

(非抽象)
na := x
      t1 t2

na1 -> t1'
----------------
na1 t2 -> t1' t2


t2 -> t2'
----------------
nanf1 t2 -> nanf1 t2'

とする。
